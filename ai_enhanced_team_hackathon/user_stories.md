# User Stories - AI Code Auditor

- **Story 1:** As a developer, I want to upload my Python script so that the AI can audit it for security flaws.
- **Story 2:** As a developer, I want to paste my SQL queries so that I can find performance bottlenecks.
- **Story 3:** As a user, I want to see a clear list of vulnerabilities so that I can prioritize what to fix first.
- **Story 4:** As a developer, I want to receive specific code suggestions for each issue so that I can fix bugs quickly.
- **Story 5:** As a user, I want to see a severity score for each issue so that I can focus on critical security risks.
- **Story 6:** As a developer, I want to export the audit result as a Markdown report so that I can share it with my team.
- **Story 7:** As a user, I want a history of my previous audits so that I can track my code improvements.
- **Story 8:** As a student, I want to see an explanation of why a specific code block is considered "bad practice" so that I can learn and avoid it in the future.
