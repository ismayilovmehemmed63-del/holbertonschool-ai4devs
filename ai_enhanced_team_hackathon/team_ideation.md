# Team Ideation - AI-Enhanced Hackathon

## Brainstormed Ideas
1. **Smart Hardware Troubleshooter:** An AI that helps diagnose BIOS and laptop hardware issues.
2. **AI Code Auditor:** A tool that performs rapid security and performance audits on Python/SQL files. ✅ (chosen)
3. **Automated Workflow Generator:** An AI that creates n8n automation pipelines from text descriptions.

## Problem Statement
Developers often overlook security vulnerabilities and inefficient SQL queries during rapid prototyping. This tool provides an immediate AI-powered audit to ensure production readiness.

## MVP Features
- **File Upload:** Support for Python (`.py`) and SQL (`.sql`) files.
- **AI Audit Engine:** Analysis of code for security flaws (e.g., SQL injection) and performance bottlenecks.
- **Remediation Report:** A structured Markdown report with "Quick Fix" suggestions.
- **Priority Scoring:** Categorizing issues into Critical, High, and Low severity.
