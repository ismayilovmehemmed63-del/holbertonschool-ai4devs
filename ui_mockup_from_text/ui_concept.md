# UI Concept - Fitness Tracker

## Purpose
To help users track their daily workouts, calorie intake, and monitor their physical progress over time.

## Users
- Casual Gym-goers: People who want to log their daily exercises.
- Professional Athletes: Users who need detailed performance metrics and history.

## Core Screens
- Dashboard: A summary of daily steps, calories burned, and active minutes.
- Workout Logger: A form to add specific exercises, sets, and reps.
- History List: A chronological list of past workouts with filters.

## Actions
- Log a new workout session.
- Set daily fitness goals.
- View and filter workout history.
- Delete or edit past entries.
