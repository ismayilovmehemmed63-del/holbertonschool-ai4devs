# User Stories - AI Study Planner

- **Story 1:** As a student, I want to enter my exam dates so that the system knows my deadlines.
- **Story 2:** As a user, I want to list my subjects so that I can organize my study materials.
- **Story 3:** As a student, I want the AI to generate a daily study plan so that I don't waste time planning manually.
- **Story 4:** As a user, I want to mark tasks as "Completed" so that I can track my progress.
- **Story 5:** As a user, I want to set the difficulty level for each subject so that the AI allocates more time to harder topics.
- **Story 6:** As a student, I want to see a countdown to my next exam so that I stay motivated.
- **Story 7:** As a user, I want to edit my study hours so that the plan adapts to my daily schedule.
- **Story 8:** As a student, I want a "Focus Mode" timer so that I can study without distractions.
- **Story 9:** As a user, I want to see a weekly summary of my study hours so that I can analyze my productivity.
- **Story 10:** As a student, I want to receive a morning notification with my tasks for the day.
